# 🌾 AI 智能农业物联网数据中间件

## 项目简介
本项目是大创项目“AI智能农业”的数据后端核心模块。负责高并发传感器数据的清洗、缓冲写入 PostgreSQL，并为 AI 算法组提供基于**时间窗口分块 (Time-Chunking)** 的高效数据查询接口。

## ✨ 核心亮点
1. **高性能批量写入**：采用内存缓冲机制，将高频 IoT 小包合并为事务提交，提升写入吞吐量。
2. **AI 友好型分块查询**：自动将连续时间序列数据切割为 **2小时** 的标准块，完美契合深度学习模型的 Batch 训练需求。
3. **时空索引优化**：针对 `time` 和 `device_id` 建立复合索引，实现毫秒级范围查询。

## 📂 模块说明
- `models.rs`: 定义数据结构及 API 交互协议。
- `processor.rs`: 原始 JSON 数据清洗与标准化。
- `db_manager.rs`: 数据库连接管理、批量写入逻辑及分块查询算法。
- `main.rs`: 演示入口，模拟从数据写入到 AI 查询的全流程。
