# 文件结构说明

## 根目录

- `README.md`：项目简介。
- `.github/`：GitHub 配置目录。
  - `workflows/`：CI 与分支保护等自动化流程。
  - `pull_request_template.md`：PR 模板。
  - `CODEOWNERS`：代码所有者配置。
- `doc/`：项目文档目录。
  - `.gitignore`：文档目录忽略规则。
  - `文件结构说明.md`：当前文件结构说明文档。
  - `贡献流程.md`：贡献流程说明文档。
- `gateway/`：网关程序（Rust）。
  - `Cargo.toml`：网关 crate 配置。
  - `Cargo.lock`：依赖锁定文件。
  - `README.md`：网关使用说明。
  - `scripts/`：运行/辅助脚本。
    - `run_mq7_gateway.sh`：MQ-7 串口模式启动脚本。
  - `src/`：网关源码。
    - `main.rs`：程序入口，仅负责参数解析与运行调度。
    - `config.rs`：命令行参数解析与运行配置。
    - `constants.rs`：默认参数与常量定义。
    - `serial.rs`：串口读取、MQ-7 数据解析与 payload 生成。
    - `gateway.rs`：UDP 发送主循环与 ACK 校验。
  - `target/`：本地构建产物目录（自动生成，可忽略）。

## 维护建议

- 新增目录或关键文件后，同步更新本文件。
- 文档命名建议使用清晰、可读的中文或中英双语。
